# React Boilerplate

React Configured Blank App

##### Stack:
  - Typescript
  - React
  - Luxon
  - Loadable Component (for code splitting)
  - Styled Components

### Installation
```sh
$ git clone https://github.com/epifanovmd/react-configured-blank-app.git
$ cd react-configured-blank-app
$ yarn
```

### Run
```sh
$ yarn dev
```
